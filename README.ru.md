# minivi

Этот проект полностью перенесен на https://altlinux.space/aleksandershad/minivi

Небольшое приложение для просмотра изображений.

Все минималистично.

Функциональные возможности:

- закрытие приложение по клавишам Esc / Q / Ctrl + Q
- открытие изображений, при ассоциации файлов, так и из приложения по клавишам Ctrl + O / O
- установка изобрежения в качестве фона рабочего стола по клавишам W
- информация об изображении I
- удаление изображения по клавише Del / D
- вызов справки по клавише F1
- переключение изображения стрелками, либо правой/левой кнопкой мыши
- увеличение/уменьшение по колесу мыши, либо +/- (может не работать в tiling)

Для поддержки работы с форматом webp в системе должен быть установлен `libwebp-tools`.
